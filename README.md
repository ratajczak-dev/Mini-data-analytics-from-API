# Mini-data-analytics-from-API
## Basic Info
The mini application retrieves data from a public API, calculates user activity statistics, and visualises them in graphs.

*User activity: To calculate user activity, I decided that activity would be the number of added post.


## Used technologies
* HTML
* CSS
* JavaScript + Chart.js
